# Alex-library-management-system

This is a C++ console-based application.

## Features
-List all books.
-list of all books by alphabet of the titles of the books.
- Add new book.
- Update an existing book.
- Remove a book.
- Borrow a book.
- Return a book.
- search for a book.
- search for all books by the first litter of the title of these books.
- Exit.

## Technologies
- C++
